# Zone-H-Grabber

Usage : python3 scan.py nick

Example : python3 scan.py chinafans




### How to install

```
git clone https://github.com/lexavey/Zone-H-Grabber
cd "Zone-H-Grabber"
python3 -m pip install -r requirements.txt
python3 scan.py chinafans
```
